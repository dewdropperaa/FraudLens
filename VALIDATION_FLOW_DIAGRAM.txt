                         Required Fields Validation
═══════════════════════════════════════════════════════════════════════════════

Each claim must have ALL 5 required elements:

┌────────────────────────────────────────────────────────────────────────┐
│  1. 👤 Patient Identity                                                 │
│     ├─ CIN (Carte Nationale)                                           │
│     ├─ Full Name                                                       │
│     └─ Patient ID / Number                                             │
│                                                                         │
│  2. 🏥 Medical Service                                                  │
│     ├─ Diagnosis                                                       │
│     ├─ Procedure / Treatment                                           │
│     └─ Service Description                                             │
│                                                                         │
│  3. 🏢 Provider/Hospital                                                │
│     ├─ Hospital / Clinic Name                                          │
│     ├─ Doctor Name                                                     │
│     └─ Medical Facility                                                │
│                                                                         │
│  4. 📅 Date of Service                                                  │
│     ├─ Service Date                                                    │
│     ├─ Consultation Date                                               │
│     └─ Hospitalization Period                                          │
│                                                                         │
│  5. 💰 Cost / Billing Amount                                            │
│     ├─ Total Amount (> 0)                                              │
│     ├─ Invoice Amount                                                  │
│     └─ Billing Total                                                   │
└────────────────────────────────────────────────────────────────────────┘

═══════════════════════════════════════════════════════════════════════════════
                            Rejection Triggers
═══════════════════════════════════════════════════════════════════════════════

Claim is INVALID if ANY of these conditions are met:

┌─────────────────────────────────────────────────────────────────────┐
│  ❌ Missing ≥ 2 Required Fields                                      │
│     Example: No patient identity + No cost = REJECTED               │
│                                                                      │
│  ❌ Document Type NOT in [medical_invoice, pharmacy_invoice,        │
│                           hospital_bill]                             │
│     Example: Prescription only = REJECTED                           │
│                                                                      │
│  ❌ Validation Score < 40                                            │
│     Example: Insufficient data = REJECTED                           │
│                                                                      │
│  ❌ Prompt Injection Detected                                        │
│     Example: "IGNORE INSTRUCTIONS" = REJECTED                       │
│                                                                      │
│  ❌ Zero or Missing Amount                                           │
│     Example: amount = 0 or null = REJECTED                          │
└─────────────────────────────────────────────────────────────────────┘
